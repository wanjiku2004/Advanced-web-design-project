This repository contains a simple Student Management System built using a LAMPP (XAMPP for Linux) environment. The system features a modern frontend web dashboard (index.php) where users can fill out a registration form to dynamically save student profiles (Names, Emails, and GPAs) without reloading the page, a secure PHP API script (add_student.php) that uses prepared statements to safely pass inputs and prevent SQL injection, and a structured MySQL database dump file (database.sql) to automatically instantiate the required table schemas locally. To launch the application, clone this repository directly inside your system's /opt/lampp/htdocs/ folder, import the database.sql file via phpMyAdmin under a new database named school_system, ensure your Apache and MySQL servers are running, and navigate to http://localhost/school-system/index.php in your web browser.

# University Portal Management System

The University Portal Management System is a role-based web application developed using PHP and MySQL to support multiple user categories, including students, lecturers, and administrators. The system provides role-based login functionality, separate dashboards for each user role, session management, and access restrictions to ensure users can only access authorized resources. Students can view academic information, lecturers can manage courses and results, and administrators can oversee system operations and user management. Built using HTML, CSS, PHP, and MySQL, the project demonstrates key concepts in authentication, authorization, role-based access control (RBAC), database integration, and secure session management. The system enhances security by restricting unauthorized access and redirecting users to the login page when necessary, making it a practical example of a multi-user web application.

# Employee Portal System

The Employee Portal System is a PHP and MySQL-based web application designed to allow employees to register, authenticate, access protected pages, and manage their accounts securely. The system includes employee registration, login authentication, a protected dashboard, a profile page, session management, and logout functionality. Developed using HTML, CSS, PHP, and MySQL, the project demonstrates essential concepts such as user authentication, secure access control, password protection, and session handling. Employee records, including name, email, department, and password, are stored in a database, and access to restricted pages is granted only to authenticated users. This project provides hands-on experience in building secure employee management portals and implementing authentication mechanisms in web applications.

# Student Portal Login System

The Student Portal Login System is a web-based application developed using PHP and MySQL that enables students to register, log in securely, access a personalized dashboard, and log out of the system. The project uses HTML, CSS, PHP, and MySQL and demonstrates fundamental web development concepts such as database connectivity, user authentication, password hashing, session management, and access control. The system stores student information including full name, email, and password in a MySQL database and restricts dashboard access to authenticated users only. Key files include dbConnect.php, register.php, login.php, dashboard.php, and logout.php. This project provides practical experience in developing secure login systems and managing user sessions within a web application environment.
